simpl.info Web Audio with RTCPeerConnection

Getting audio...

An audio source provided by WebAudio is encoded and transmitted via PeerConnection, then played out via an audio element.

Press Call and listen to the Opus-encoded audio file.

Press Drum to mix in a short drum sound

For more information, see Getting Started With WebRTC and Getting started with the Web Audio API.

The code for this example was adapted from a demo by Henrik Andreasson.
