Miscellaneous Tests

  • Test to see if denormals are flushed to zero. Currently the test is set so that it always fails so that the last non-zero value is printed. For chrome with denormals flushed to zero, the last non-zero value should be at index 139. When denormals are not flushed, the last index is larger, possibly up to 200 or more.
  • Test ogg file decoding, for android. Look at the console log to see if the decoded length is correct. (See crbug 382271.)
  • Test decoding time. Just a simple test to see how long it takes decodeAudioData to decode an m4a file.
